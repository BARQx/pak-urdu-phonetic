Pak Urdu Phonetic is an Urdu keyboard layout faithfully ported from the popular Windows-based Pak Urdu Installer layout by mBilalm. It offers the same simple, intuitive phonetic typing experience and brings this widely adopted standard to the Keyman platform for cross-platform use.

With over 160+ unique key mappings across regular, Shift, AltGr (Right Alt), and Shift+AltGr combinations, this layout offers the most complete phonetic Urdu typing solution available — more comprehensive than any previous Urdu keyboard layout to date.

📥 Installation

🖥️ Via Keyman (Windows, macOS, Linux)

🐧 Linux (Ubuntu/Debian)

sudo add-apt-repository ppa:keymanapp/keyman
sudo apt update
sudo apt install keyman

Other Linux distributions: Download from keyman.com/linux

🪟 Windows

Download and install Keyman for Windows

🍎 macOS

Download and install Keyman for macOS

📥 Install the Urdu Keyboard

  1. Visit the Pak Urdu Phonetic keyboard page
  2. Download the pak_urdu_phonetic.kmp package
  3. Double-click the downloaded file to open it in Keyman
  4. Click Install when prompted

⌨️ Switch to the Keyboard

Open Keyman from your system tray (Windows/Linux) or menu bar (macOS), and select Pak Urdu Phonetic from the keyboard list.

🐧 Via XKB (Linux — Ubuntu 26.04 LTS and later)

Pak Urdu Phonetic is now available as a built-in keyboard layout on Ubuntu 26.04 LTS and other latest distributions. No third-party software required.

Open Settings → Keyboard → Input Sources, search for Pak Urdu Phonetic, and add it.
